## 特性

- **地址栏直达搜索**：自定义地址栏搜索引擎后即可输入内容直接回车，即可自动填充到豆包网页版并发送并附加模式提示词，实现搜索引擎体验
- **全局菜单**：任意页面点击扩展菜单，选择喵模式快速提问
- **六种专属喵模式**：
  - 🔍 **喵搜模式**（默认）—— 强大搜索引擎风格，附带提供网页链接
  - ⚡ **一句喵模式** —— 极简快速，只输出核心答案
  - 📚 **笨喵模式** —— 超耐心小白教学，从零开始深入浅出
  - 🎓 **学喵分析** —— 严谨学术风格
  - ⚖️ **喵对比** —— 决策辅助，对比优缺点
  - 🌐 **喵翻译/喵润色** —— 专业翻译与写作助手
- **防幻觉**：每种模式都加入严格提示词，避免乱编
- **喵交互**：喵～你要问什么呢？

## 用法：把喵搜设为浏览器默认搜索引擎

这样你就可以像用 Google 一样，直接在地址栏打字回车让喵搜回答！

###  Edge 为例

1. 打开 **设置 → 隐私、搜索和服务／搜索和连接体验／地址栏和搜索/搜索引擎 → 添加搜索引擎**
2. 点击“添加”
3. 填写：
   - 搜索引擎名称：`喵搜`
   - 关键字：`nya`
   - URL：  
     ```
     https://www.doubao.com/chat?prompt=%s
     ```
4. 添加后，选择设为默认值
5. 使用方式：
   - 直接在地址栏输入问题 → 回车
   - 指定关键字触发：输入 `nya` + 空格 + 问题 → 回车
